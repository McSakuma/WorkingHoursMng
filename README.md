##システム名
WEB工数管理システム(WEB Working Hours Management System)

##システム概要
作業工数を管理するシステムです。  
作業者に毎日の工数について登録してもらいます。  
社員毎の毎月の作業状況の確認ができます。  
プロジェクト毎の月別の工数状況やTOTALの作業工数の確認ができます。
